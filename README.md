## ğŸ› Troubleshooting

**Build fails?**
- Ensure all dependencies are installed
- Check Python version (3.8+ required)
- Try: `buildozer android clean` then rebuild

**App crashes on launch?**
- Grant all permissions in Android Settings
- Check Android version compatibility
- View logs: `adb logcat | grep python`

**AutoGlance not working?**
- Grant camera permission
- Ensure front camera is accessible
- Check good lighting conditions

**Files not showing?**
- Grant storage permissions
- Wait for file scanning to complete
- Check if files are in accessible locations
